Pattern Matching Engine

The Intel® Curie™ powered microcontroller contains 128 parallel Processing Elements (PE), useful for learning and recognising patterns without requiring the main CPU. This feature provides an energy efficient process of learning and responding to complex events.

For further information and example code, refer to the Intel Open Source Technology Center repository on GitHub
